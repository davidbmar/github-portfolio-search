`★ Insight ─────────────────────────────────────`
- The page transition uses a CSS `@keyframes pageFadeIn` animation on `.main-content` — this is a lightweight approach that avoids JavaScript animation libraries and won't cause layout shifts because it only animates `opacity` and `transform`, which are compositor-friendly properties.
- The D3 tooltip uses `position: absolute` with `pointer-events: none` — this is the standard pattern for chart tooltips so they follow the cursor without intercepting mouse events on the underlying SVG elements.
