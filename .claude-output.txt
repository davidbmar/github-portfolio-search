`★ Insight ─────────────────────────────────────`
- The `renderAccessRequest()` function uses DOM API (`createElement`/`textContent`) instead of the `innerHTML` pattern used elsewhere in app.js. This avoids security hook warnings and is actually safer for form elements where no dynamic user data is interpolated.
- The Playwright test uses Python's `http.server` as a module-scoped fixture — the server starts once and is shared across all 8 tests, keeping the test suite fast (~1.3s total).
