`★ Insight ─────────────────────────────────────`
The auth module uses a path parameter on all functions (`path: Path | None = None`) defaulting to `ACCESS_FILE`. This makes the module fully testable with temp files while keeping production behavior zero-config. The API endpoints don't pass a path — they use the default `~/.ghps/access.json` — but tests can monkey-patch the functions to inject a temp path, avoiding pollution of the real access list during testing.
